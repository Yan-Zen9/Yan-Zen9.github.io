# Yan Zen Academic Homepage

Personal academic homepage built with [al-folio](https://github.com/alshedivat/al-folio) and GitHub Pages.

## Main Pages

- Home: `_pages/about.md`
- Publications: `_pages/publications.md` and `_bibliography/papers.bib`
- CV: `_pages/cv.md` and `_data/cv.yml`

## Local Preview

```bash
export PATH="/opt/homebrew/opt/ruby/bin:$PATH"
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
